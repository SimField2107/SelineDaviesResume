Minor Figures

Notorious Penny Design Game

  • Client brief: an interactive disguise-creation game for integration into the Minor Figures Shopify site. Delivered as a complex site and then stripped of dependencys and eventually packaged as a flat game for integration with the minor figures website.
  • Project date: August 2024
  • Project URL: https://notorious.minorfigures.com/

Built with: JavaScript · Adobe PhotoShop · GSAP · Bootstrap · HTML5 · SCSS · Twig ·

- Custom disguise-builder game with drag-and-drop interactivity.
- Dual user paths: “Play for Fun” mode and “Play to Win” with automated receipt validation.
- GSAP animations and Bootstrap for Shopify integration.
- Image capture and download using HTML-to-canva.
- Fully mobile-first and desktop-compatible, with smooth UX and strong brand identity.
- Optimized assets, removed Twig and unnecessary JS, enabling lightweight deployment on the client’s Shopify site.
