Minor Figures

Notorious Penny Design Game

  • Client brief: An interactive disguise-creation game for integration into the Minor Figures Shopify site.
  • Project date: August 2024
  • Project URL: notorious.minorfigures.com
Built with: JavaScript · GSAP · Bootstrap · SCSS
  • Custom disguise-builder game with drag-and-drop interactivity.
  • Dual user paths: “Play for Fun” and “Play to Win” with receipt validation.
  • GSAP animations and Bootstrap for Shopify integration.
  • Image capture and download using HTML-to-canvas.
  • Optimized assets for lightweight deployment on the client’s Shopify site.
