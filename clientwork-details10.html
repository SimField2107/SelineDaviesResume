Coca-Cola

Co-op Meals that Matter

  • Client brief: Develop an accessible, single-page microsite for a joint Co-op and Coca-Cola promotion, in partnership with Special Olympics Great Britain.
  • Project date: April 2025
  • Project URL: https://www.cokemealsthatmatter.co.uk/
Built with: Adobe Photoshop · HTML5 · SCSS · JavaScript
  • Developed a fully responsive, single-page microsite from Adobe Photoshop designs.
  • Prioritized accessibility, implementing semantic HTML5 and comprehensive ARIA attributes for screen readers.
  • Ensured all interactive elements were keyboard-navigable with clear focus indicators.
  • Implemented carousels with pause/play controls and reduced motion settings to meet WCAG guidelines.
  • Conducted thorough accessibility audits to ensure an inclusive user experience.
