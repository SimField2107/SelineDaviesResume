Dolmio

Chance to win £10,000

  • Client brief: Develop a high-engagement promotional microsite for a major Dolmio prize draw. The site needed to manage user entries, handle proof-of-purchase photo uploads, and integrate with the main brand website.
  • Project date: August 2025
  • Project URL: https://www.dolmio.co.uk/win
Built with: Photoshop · HTML5 · SCSS · JavaScript · PHP · Symfony
  • Developed a promotional microsite from Photoshop designs, ensuring a pixel-perfect and brand-aligned experience across all desktop and mobile devices.
  • iIntegration into the main Dolmio website, scoping all CSS and JavaScript to operate within a reverse-proxied environment that shared the live site's header and footer.
  • Crafted decorative page elements, purely with SCSS to reduce HTTP requests, improve load times, and ensure crisp, scalable visuals.
  • Styled the complete user journey, including the entry form with image upload functionality, dynamic win/loss confirmation screens, and an interactive FAQ overlay.
  • Ensured the final build was optimized for high-traffic promotional periods and integrated perfectly with the Symfony backend for secure prize draw management.
