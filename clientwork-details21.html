The Jammy Drop - Plinko browser game

  • Client brief: Develop a physics engine based Plinko browser game for a major promotional campaign with multiple prize routes and coupon print flow.
  • Project date: September 2025
  • Project URL: https://www.thejammydrop.co.uk/
Built with: Figma · HTML5 · SCSS · JavaScript · GSAP · matter.js
  • Engineered the game's physics environment using Matter.js, precisely defining the world boundaries and programmatically generating all static objects, including pegs and bumpers, from coordinate arrays.
  • Architected a complex, multi-state user journey, handling conditional routing for win/loss scenarios and prize claims.
  • Developed and styled the complete user flow, including interactive modals, animated win screens with a JavaScript confetti library, HTML emails.
  • Implemented a custom-styled, cross-browser compatible print dialogue screen for prize coupons.
