AR 'Spin To Win' Campaign

  • Client brief: An AR-powered campaign where users scanned a QR code on-bottle to launch an augmented reality prize wheel.
  • Project date: July 2024
  • Highlights: 293,000+ entries & a 14.8% uplift in sales.
Built with: JavaScript · HTML5 · CSS
  • Developed an AR-powered microsite accessible via QR codes on 1.68 million bottles.
  • Integrated an immersive, camera-tracked prize wheel experience.
  • Built a “Stick or Twist” mechanic to encourage strategic user participation.
  • Implemented email verification, IP recognition, and CAPTCHA to reduce fraud.
  • Integrated an AI chatbot, resulting in a 63% drop in consumer queries.
