Diet Coke x Boots

Chance to win 1 of 50 Boots eGift Cards

  • Client brief: Develop a promotional microsite for a Diet Coke x Boots partnership, featuring a daily prize draw. Key requirements included a secure, multi-stage user entry process with email verification to ensure valid participation.
  • Project date: August 2025
  • Project URL: https://dietcokeprize.co.uk/
Built with: Photoshop · HTML5 · SCSS · JavaScript · PHP · Symfony
  • Constructed a pixel-perfect, fully responsive frontend from Photoshop designs, ensuring the campaign's visual identity was consistent across all devices.
  • Architected a multi-step user journey, including an entry form with reCAPTCHA, a secure email verification stage with a timed code, and dynamic content for promotion end states.
  • Designed and built a suite of responsive HTML emails for the two-factor authentication process, ensuring they rendered correctly across all major email clients.
  • Implemented key UI components, including a GDPR-compliant cookie consent banner and an accessible modal window for displaying the privacy policy.
  • Integrated the frontend with the Symfony backend to handle secure data submission and manage the logic for the prize draw's daily entry mechanic.
