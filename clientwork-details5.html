Skittles Poundland Giant Gooey voting promotion

  • Client brief: Skittles Gooeys Poundland campaign, users scan a QR code in-store to vote for their favorite Skittles variety and instantly win cash prizes.
  • Project date: August 2024
  • Project URL: https://www.skittles.co.uk/poundland/scan-vote-win

Built with: Adobe XD · Illustrator · GSAP · HTML5 · SCSS · Javascript

- Responsive Registration Flow: Custom form and DOB age validation.
- Dynamic Voting Module: Vote between Skittles product types, with real-time results visualized as a live-updating bar chart.
- Win/Non-Win Animations: Custom animated outcome pages based on win logic.
- Prize Claim Page: Secure form for UK bank details with unique claim link support.
- Admin Tooling: Searchable entry database and downloadable prize and voting reports.
- API Integration: Integrated with Mars’ API for data submissions.
- Cross-Domain Embedding: Site built to be hosted within an iframe and cookie banner.
- Accessibility & Compliance: Followed Mars’ brand standards and privacy policies.
