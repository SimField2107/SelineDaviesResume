Tour of Terlato

  • Client brief: Develop a responsive microsite with a smooth, animated carousel component show casing the wines of Terlato.
  • Project date: August 2024
  • Project URL:

Built with: JavaScript · Adobe PhotoShop · HTML5 · CSS Sprites

- Created custom bottle carousel using JavaScript and CSS transitions, with fade-in/fade-out effects.
- Implemented automatic slide cycling with timed intervals and pause-on-hover functionality.
- Built manual navigation controls (next/prev arrows and clickable indicators) with accessible focus states.
- Used CSS keyframe animations to handle smooth opacity transitions between bottle images.
- Ensured all carousel content was mobile-first and optimized for touch interaction and swipe support.
- Included lazy loading logic to improve performance by loading images as they enter view.
- Structured carousel to allow easy updates via HTML data attributes for each slide.
- Added fallback styles and no-JS message for users with disabled scripts.
- Performed browser/device testing to validate animation smoothness and layout responsiveness.
