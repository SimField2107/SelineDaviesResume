A Trip to the Amalfy Coast Draw

  • Client brief: Developed the front-end for an offline prize draw promotion microsite for Malfy Gin. Consumers entered by either uploading a receipt or submitting a unique on-pack code, for a chance to win a luxury holiday to the Amalfi Coast.
  • Project date: April 2025
  • Project URL:

Built with: Figma · HTML5 · SCSS · Javascript· Twig · PHP

- Built responsive entry form with support for both unique code and receipt upload methods.
- Integrated conditional logic for form validation and field visibility based on user input type (code vs receipt).
- Developed custom receipt upload UI with upload guidelines modal.
- Designed and styled all key screens including entry form, thank you page, and fallback/error states.
- Age gate, cookie notice, and email confirmations.
- Delivered the site as a fully responsive iframe embed for integration into the Malfy Gin brand website.
- Structured front-end to allow integration with 40,000 pre-generated codes and receipt tracking without OCR.
- Performed cross-browser and device testing to ensure consistency across all modern platforms.
